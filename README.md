For the G25 collections:

YFull naming convention was used for the following Y-DNA haplogroups:
-	C2b
-	E-M34
-	G
-	I2 (with the exception of L596)
-	N
-	Q (other than Native American branches)
-	R1a
-	R1b-PF7562
-	R1b-Z2103

All other lineages + mentioned exceptions use the FTDNA naming. For Y-DNA lines named after YFull some terminal branches not mentioned there were added from FTDNA. For certain East Asian branches terminal branches not mentioned in either platform  were added from theytree. Some after YFull named  branches have extra intermediary mutations from  FTDNA.
Please contact me via email if you have noticed mistakes in this regard.
